📖 المصحف التفاعلي

نظام مصحف تفاعلي متطور يستخدم صفحات SVG مع بيانات KFGQPC وطبقة تفاعل ذكية للتنقل والتفاعل مع آيات القرآن الكريم

الميزات الرئيسية

  • عرض صفحات المصحف بصيغة SVG عالية الجودة
  • تفاعل ذكي مع الآيات عند النقر عليها
  • قياس دقيق لعرض النص باستخدام الخط العثماني
  • تصفح سريع بين الصفحات
  • واجهة مستخدم عربية متجاوبة
  • دعم كامل للكتابة من اليمين لليسار

🚀 صفحة تجريبية

تجربة كاملة للمصحف التفاعلي مع إمكانية التنقل بين الصفحات والتفاعل مع الآيات

📄 صفحة محددة

عرض صفحة معينة من المصحف (مثال: صفحة 3) مع إمكانية التفاعل مع الآيات

🔗 API

واجهة برمجية للحصول على بيانات آيات صفحة معينة بصيغة JSON

⚙️ لوحة الإدارة

إدارة البيانات والآيات والصفحات من خلال لوحة إدارة Django
